# Master File Manager
## Este es un proyecto para hacer operaciones tediosas con archivos.

- Puedes seleccionar un directorio y guardar en un documento los nombres de todos los archivos que contenga. 
Por ejemplo si quieres entregarle a una IA libros de programación para que compare y te recomiende el mejor.
- Puedes crear carpetas y archivos masivamente. Si deseas por ejemplo crear 10 carpetas para guardar los videos de 
tus 10 artistas favoritos de manera separada, puedes crearlas escribiendo los nombres de las carpetas para luego
generarlas. Lo mismo con archivos.
- De la misma manera, eliminar archivos. Pasa los nombres y listo.
