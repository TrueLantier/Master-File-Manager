# Notas de las clases CrearArchivos y CrearCarpetas.
## ¿Qué pueden hacer?

### Función básica: Crean carpetas y archivos.

- Estas clases pueden crear carpetas o archivos masivamente. Ambos pueden ser con nombres predeterminados o personalizados.

- Se pueden crear carpetas anidadas. 

- Si los objetos a crear existen, se da la posibilidad de sobreescribirlos o no.

- Se puede pasar un array con los nombres de los archivos o carpetas a crear.
