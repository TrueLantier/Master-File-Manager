# Notas de la clase Crear Archivos.
## ¿Qué puede hacer?

### Función básica: Crear carpetas y archivos.

- Esta clase puede crear carpetas y archivos masivamente. Ambos pueden ser con nombres específicos o personalizados.

- Se pueden crear carpetas anidadas. Si los objetos a crear existen, se da la posibilidad de sobreescribirlos o no.

- Se puede pasar un array con los nombres de los archivos o carpetas a crear.
